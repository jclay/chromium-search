- Single-file Python 3.11+ tool (`chromium-search.py`), run with `uv run chromium-search.py`
- Zero dependencies — stdlib only
- Keep it simple: no build system, no packaging

## Dev commands

- Format: `uvx ruff format chromium-search.py`
- Lint: `uvx ruff check chromium-search.py`
- Typecheck: `uvx ty check chromium-search.py`
